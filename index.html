Installation Guide

安装教程（图文） Step-by-step install guide

①
下载并解压
Download & Unzip

点击下方按钮下载 zip 安装包 → 右键 zip 文件 → 「解压到此处」→ 得到一个文件夹

②
开启开发者模式
Enable Developer Mode

打开 Chrome → 地址栏输入 chrome://extensions/ → 右上角打开「开发者模式」开关

③
加载插件文件夹
Load Unpacked

点击「加载已解压的扩展程序」→ 选择刚才解压的文件夹 → 插件图标出现在工具栏 ✅

④
填入 API Key
Enter Your Key

点击插件图标 → 粘贴你的 API Key → 点「保存」→ 进入设置选择对应的 AI 提供商

⑤
开始使用
Start Using

确认插件开关为「ON」→ 在任意网页选中英文 → 弹窗自动出现 → 悬停单词看词卡 🎉
